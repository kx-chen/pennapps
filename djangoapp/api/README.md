# Interview App REST API

### Rest Framework
* `pip install djangorestframework`

### Quickstart
* `cd djangoapp`
* `python manage.py runserver`
* Visit `http://127.0.0.1:8000/api/` to view api call protocols
* Visit `http://127.0.0.1:8000/admin/` to view databases
